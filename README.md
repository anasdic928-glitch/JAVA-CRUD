Deskripsi Proyek

JavaCRUD adalah aplikasi desktop berbasis Java Swing yang digunakan untuk mengelola data siswa. Aplikasi ini menerapkan konsep CRUD (Create, Read, Update, Delete) sehingga pengguna dapat menambah, melihat, mengubah, dan menghapus data siswa yang tersimpan pada database MySQL.

Tujuan Program

1.Mengelola data siswa secara digital.
2.Menerapkan konsep CRUD pada Java.
3.Menghubungkan aplikasi Java dengan database MySQL.
4.Melatih penggunaan Java Swing untuk membuat GUI.
Struktur Proyek

JavaCRUD/ ├── pom.xml └── src/main/java/com/mycompany/javacrud/ ├── Main.java ├── FormSiswa.java └── Koneksi.java

Kesimpulan

Aplikasi JavaCRUD merupakan aplikasi desktop berbasis Java Swing yang menerapkan operasi CRUD (Create, Read, Update, Delete) pada data siswa. Program ini memanfaatkan JDBC untuk menghubungkan aplikasi dengan database MySQL dan menggunakan Maven untuk mengelola dependency proyek. Dengan adanya aplikasi ini, proses pengelolaan data siswa dapat dilakukan secara lebih cepat, mudah, dan terstruktur
